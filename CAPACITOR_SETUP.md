# Capacitor Mobile Setup Guide

Your app is now ready for mobile deployment with Capacitor!

## Prerequisites

### For Android:
- Install [Android Studio](https://developer.android.com/studio)
- Set up Android SDK and emulator (or connect a physical device)

### For iOS (Mac only):
- Install [Xcode](https://apps.apple.com/us/app/xcode/id497799835?mt=12)
- Install CocoaPods: `sudo gem install cocoapods`

## Development Workflow

### Build and Deploy to Android:
```bash
npm run cap:android
```
This will:
1. Build your web app
2. Copy assets to Android project
3. Open Android Studio

Then in Android Studio:
- Select your device/emulator
- Click "Run" (green play button)

### Build and Deploy to iOS:
```bash
npm run cap:ios
```
This will:
1. Build your web app
2. Copy assets to iOS project
3. Open Xcode

Then in Xcode:
- Select your device/simulator
- Click the Play button to build and run

## Updating After Changes

After making changes to your React code:

```bash
npm run cap:sync
```

This rebuilds the web app and syncs it to both platforms.

## Project Structure

- `dist/` - Built web app (generated by `npm run build`)
- `android/` - Android native project
- `ios/` - iOS native project
- `capacitor.config.ts` - Capacitor configuration

## Useful Commands

- `npm run build` - Build web app only
- `npx cap copy` - Copy web assets to native projects
- `npx cap sync` - Full sync (copy + update plugins)
- `npx cap open android` - Open Android Studio
- `npx cap open ios` - Open Xcode

## Troubleshooting

**Android build fails:**
- Make sure Android SDK is installed via Android Studio
- Check that `ANDROID_HOME` environment variable is set

**iOS build fails:**
- Run `pod install` in the `ios/App` directory
- Make sure you have the latest Xcode command line tools

**App not updating:**
- Always run `npm run cap:sync` after code changes
- Clear app cache in device settings if needed
